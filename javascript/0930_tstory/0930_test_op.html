01. 함수와 매개변수를 이용한 형태

func에 변수를 저장 하였을 때, 다시 매개변수를 통해 값을 저장해 출력할 수 있었습니다.

function func(str1, str2){
    document.write(str1 + "와" + str2 + "는 귀엽다." )
}

func("강아지","고양이");
func("햄스터", "토끼")
