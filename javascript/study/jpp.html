join()/ push()/ pop()

01. join()

join()은 배열의 사본을 반환하며, 배열의 모든 요소를 연결해 하나의 문자열로 만듭니다.

아래는 예시입니다.

    //join()을 사용하는 경우   
    const arrNum = [100,200,300,400,500];
    const text1 = arrNum.join('')
    const text2 = arrNum.join('-')

    document.querySelector.innerHTML = text1   //100200300400500가 나옴
    document.querySelector.innerHTML = text2  //100-200-300-400-500가 나옴

02. push()

push()은 배열의 마지막에 새로운 요소를 추가하고, 변경된 배열의 길이를 반환하는 함수입니다.

아래는 예시입니다.

    //push()을 사용하는 경우   
    const arrNum = [100,200,300,400,500];
    const arrPush = arrNum.push(600);

    document.querySelector.innerHTML = arrPush // 6이 나옴 //600을 추가하고 길이를 반환(숫자를 세기) 때문
    document.querySelector.innerHTML = arrNum  //100,200,300,400,500,600이 나옵니다. //num에는 push때문에 600이 추가되어 있다.

03. pop()

pop()은 배열의 마지막에 요소를 제거하고, 삭제된 요소를 반환하는 함수입니다.

아래는 예시입니다.

    //pop()을 사용하는 경우   
    const arrNum2 = [100,200,300,400,500];
    const arrPush2 = arrNum.push();

    document.querySelector.innerHTML = arrPush2 // 500이 나옴 // 마지막 요소인 500을 삭제하고 반환하였기 때문
    document.querySelector.innerHTML = arrNum2  //100,200,300,400이 나옵니다. //num에는 pop때문에 500이 삭제되어 있다.

> 정리

이름 특징 사용법
join() 배열의 사본(()안에 들어있는)을 반환 + 하나의 문자열을 만든다. const text1 = arrNum.join('')
push() 마지막에 새로운 요소(()안에 들어있는)를 추가 + 길이(갯수)를 반환 const arrPush = arrNum.push(i);
pop() 마지막 요소를 제거 + 삭제된 요소를 반환 const arrPush2 = arrNum.push();
