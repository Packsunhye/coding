indexOf()를 이용하여 검색하기

HTML 태그 검색하기

  • a : a 태그는 다른 페이지 이동을 설정합니다.
  • abbr : abbr 태그는 줄임말이나 머리글자로 표현하는데 사용합니다.
  • acronym : acronym 태그는 줄임말이나 머리글자로 표현하는데 사용합니다.
  • address : address 태그는 문서나 글의 저자 또는 회사와 연락할 수 있는 정보를 명시할 때 사용합니다.
  • applet : applet 태그는 문서에 포함되는 애플릿(웹 페이지에 포함되어 작은 기능을 수행하는 프로그램)을 정의할 때 사용합니다.
  • area : area 태그는 이미지 맵(image-map)에서 하이퍼링크가 연결될 영역을 정의할 때 사용합니다.
  • article : article 태그는 문서, 페이지, 애플리케이션, 또는 사이트 안에서 독립적으로 구분해 배포하거나 재사용할 수 있는 구획을 나타냅니다.
  • audio : audio 태그는 문서에 소리 콘텐츠를 포함할 때 사용합니다.
  • aside : aside 태그는 문서의 주요 내용과 간접적으로만 연관된 부분을 나타냅니다.
  • b : b 태그는 다른 부가적인 목적 없이 단순히 굵게 표현되는 텍스트를 정의할 때 사용합니다.
  • base : base 태그는 해당 문서의 모든 상대 주소(relative URL)에 대한 기본 URL(base URL)과 target 속성값을 정의할 때 사용합니다.
  • bdi : bdi 태그는 BDI는 Bi-Directional Isolation을 의미하며, 주위의 텍스트들과는 다른 방향으로 써지는 텍스트 영역을 정의할 때 사용합니다.
  • bdo : bdo 태그는 BDO는 Bi-Directional Override를 의미하며, 자식 요소의 텍스트 방향성(text directionality)을 정의할 때 사용합니다.
  • bgsound : bgsound 태그는 페이지가 사용되는 동안 배경에서 재생할 사운드 파일을 설정합니다.
  • blink : blink 태그는 깜박이는 텍스트로 만들어 줍니다.
  • blockquote : blockquote 태그는 포함 된 텍스트가 확장 인용 있음을 나타냅니다.
  • body : body 태그는 HTML문서의 내용을 나타냅니다.
  • br : br 태그는 텍스트 안에 줄바꿈(캐리지 리턴)을 생성합니다.
  • button : button 태그는 클릭 가능한 버튼을 나타냅니다.
