indexOf()를 이용하여 검색하기

javascript 태그 검색하기

  • indexOf() : indexOf() 메서드는 문자열에서 특정 문자의 위치를 찾습니다.
  • slice() : slice() 메서드는 문자열에서 원하는 값을 추출합니다.
  • substring() : substring() 메서드는 문자열에서 원하는 값을 추출합니다.
  • substr() : substr() 메서드는 문자열에서 원하는 값을 추출합니다.
  • split() : split() 메서드는 문자열에서 원하는 값을 배열로 바꾸어 추출합니다.
  • trim() : trim() 메서드는 문자열 앞뒤 공백 제거한 뒤 출력합니다.
  • trimStart() : trimStart() 메서드는 문자열 앞 공백 제거한 뒤 출력합니다.
  • trimEnd() : trimEnd() 메서드는 문자열 뒤 공백 제거한 뒤 출력합니다.
  • toUpperCase() : toUpperCase() 메서드는 문자열을 대문자로 변경한 뒤 출력합니다.
  • toLowerCase() : toLowerCase() 메서드는 문자열을 소문자로 변경한 뒤 출력합니다.
